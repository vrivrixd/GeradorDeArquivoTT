# Gerador De Arquivo TT
Um programa para gerar arquivos de servidores do TeamTalk.
Inclui uma versão gui em [AutoHotkey](https://autohotkey.com).
## Forma de Uso
Ao executar o programa, será solicitado os dados do servidor, conta a ser usada, apelido (opcional) e canal onde o usuário deve entrar (opcional).
Em seguida, será salvo o arquivo na pasta especificada, com o nome especificado.
## Executando da Fonte
Você precisa do [NVGT (Non Visual Game Toolkit)](https://nvgt.gg),
ou do [AutoHotkey](https://autohotkey.com),
tanto para executar, quanto para compilar.
## Compilação
Para compilar no NVGT, dê aplicações ou o botão direito do mouse no arquivo .nvgt, escolha Compile Script, a plataforma que você quer compilar e se você quer a versão Debug ou Release.
Para compilar no AutoHotkey, dê aplicações ou o botão direito do mouse no arquivo .ahk, escolha Compile script, ou se preferir, Compile script (GUI).
A opção Gui te permite personalizar as opções para a compactação do arquivo .exe e arquitetura.
